PageDAO Hub API

Welcome to the PageDAO Hub API. Below are the available endpoints:

Token Price

Get the current price of the PAGE token.

Endpoint: /api/v1/token/price

Method: GET

Example: /api/v1/token/price

Chain-Specific Token Price

Get the PAGE token price on a specific chain.

Endpoint: /api/v1/token/price/:chain

Method: GET

Example: /api/v1/token/price/ethereum

Total Value Locked (TVL)

Get the total value locked across all chains.

Endpoint: /api/v1/liquidity/tvl

Method: GET

Example: /api/v1/liquidity/tvl

Chain-Specific TVL

Get the TVL on a specific chain.

Endpoint: /api/v1/liquidity/tvl/:chain

Method: GET

Example: /api/v1/liquidity/tvl/ethereum
